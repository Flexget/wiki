=== Lists ===

- value 1

=== Indentation counts ===

{{{
option 1: value 1
option 2: value 2
option 3: value 3
}}}

{{{
option 1:
  option 2: value 2
  option 3: value 3
}}}

So what's the difference? In last {{{option 2}}} and {{{option 3}}} belongs to {{{option 1}}}.
