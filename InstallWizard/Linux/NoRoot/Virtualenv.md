= Installing !FlexGet in a virtualenv =

== What is virtual env ? ==

Virtual env is isolated Python environment. Libraries and applications inside it don't affect system installation in any way.

=== Create virtualenv ===

{{{
virtualenv ~/flexget/
}}}

=== Activate virtualenv ===

{{{
source ~/flexget/bin/activate
}}}
