= Output Statistics =

Output graphs about downloaded entries. The output is a single HTML file with graphs linked from Google chart API.

NOTE: Must use an absolute path

'''Example:'''

{{{
global:
   statistics: 
      file: /home/jdoe/public_html/flexget_stats/index.html
}}}
