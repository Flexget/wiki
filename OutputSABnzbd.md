= SABnzbd =

=== Example ===

{{{
sabnzbd:
  apikey: 123456
  url: http://localhost/sabnzbd/api?
  category: movies
}}}

'''Note:''' url has default value of {{{http://localhost:8080/sabnzbd/api?}}}

=== All possible parameters ===

{{{
sabnzbd:
  apikey: ...
  url: ...
  category: ...
  script: ...
  pp: ...
}}}
