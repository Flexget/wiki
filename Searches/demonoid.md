# Demonoid Search plugin
A search plugin that generates sorted from demonoid.

## Config Format

```
Will accept:
      demonoid: <category>

    Or:
      demonoid:
        - category: <category>
        - quality: <quality>
        - sub-category: <category>

    Categories:

    * all
    * applications
    * audio books
    * tv
    * games
    * books
    * comics
    * anime
    * misc
    * movies
    * music
    * music videos
    * pictures
```
