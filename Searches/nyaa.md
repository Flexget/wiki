# Nyaa
Search plugin which gives results from [nyaa.si](http://nyaa.si/)
### Config
Takes two options:

**Category:**

Valid choices are `all`, `anime`, `anime eng`, `anime non-eng`, and `anime raw`

**Filter:**

Valid choices are `all`, `filter remakes`, `trusted only`, and `a+ only`



### Example
```
nyaa:
  category: anime eng
  filter: trusted only
```
